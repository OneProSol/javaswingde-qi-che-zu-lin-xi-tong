### 一、相关技术
- 前端：`javaswing`。
- 后端：`java`等。
- 数据库：`MySQL`

### 二、开发软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
### 三、部署所需环境

- 后端：`jdk1.8`。
- 数据库：`mysql5.7以上`

### 四、功能描述及功能图展示
系统分为：用户、管理员。

用户功能：1. 登录、2. 注册、3. 可租用汽车、4. 汽车信息浏览、5. 我租的车

管理员功能：1. 登录、2.汽车信息录入、3. 汽车信息浏览
